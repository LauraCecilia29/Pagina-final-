7. Altium

Es un software profesional utilizado en la industria para el diseño de circuitos impresos. Ofrece un ecosistema integrado con herramientas de simulación, verificación de diseño y características avanzadas de enrutado automático. A continuación, se presenta el flujo de trabajo completo desde la creación del proyecto hasta la generación de archivos de fabricación.

El primer paso es crear un nuevo proyecto en Altium. Se selecciona File → New → Project → PCB Project y se define el nombre y ubicación del proyecto. Altium organiza el trabajo mediante un sistema de proyectos (.PrjPcb) que integra todos los archivos relacionados: esquemáticos, PCB, librerías y documentos de salida.

Para la gestión de componentes, Altium incluye acceso a librerías en línea mediante Manufacturer Part Search, que permite buscar componentes con toda su información técnica, modelos 3D y disponibilidad comercial. Se accede desde Panels → Components y se busca por número de parte o descripción. Cada componente integra símbolo esquemático, huella PCB y modelo 3D.

Para el diseño de un nuevo esquemático se da click derecho en el proyecto → Add New to Project → Schematic. Para colocar componentes se usa la tecla P, se pueden rotar con la barra espaciadora durante la colocación y copiar múltiples con Ctrl+D.

Las conexiones se realizan mediante cables (Ctrl+W) que conectan punto a punto. Para limpiar visualmente el esquemático se pueden usar Net Labels (P → N) que etiquetan redes sin cables físicos. Altium conecta automáticamente todos los puntos con el mismo nombre de red, incluso entre múltiples hojas. También se utilizan Power Ports (P → Shift+P) para VCC, GND y otras señales de alimentación. Las etiquetas (Net Labels) son fundamentales para mantener el diseño ordenado. Se colocan en las conexiones y permiten que el esquemático sea legible sin necesidad de cables que atraviesen toda la página.

El primer paso es encender el soldador y esperar a que llegue a la temperatura adecuada (entre 300º y 400º). Una vez que lo haga, se estaña la punta (se recubre con una pequeña cantidad de soldadura) para protegerla y ayudar con la transferencia de calor. Despues, se limpia la punta con una esponja húmeda para quitar cualquier exceso de soldadura u oxidación.

Para transferir el diseño al PCB se crea un nuevo documento PCB (click derecho en proyecto → Add New to Project → PCB) y se ejecuta Design → Update PCB Document. El panel ECO (Engineering Change Order) muestra los cambios a aplicar. Después de validar con Validate Changes se ejecutan con Execute Changes.

Se configura el contorno de la placa usando Design → Board Shape y se definen las reglas de diseño en Design → Rules. Las reglas principales incluyen Clearance (separación mínima entre pistas), Width (ancho mínimo de pistas) y Via Style (tamaño de vías). Altium verifica reglas en tiempo real y las violaciones aparecen como líneas verdes en el diseño.

Los componentes se organizan usando herramientas como Tools → Component Placement → Arrange Within Rectangle para auto-posicionamiento, o las opciones de alineación y distribución del menú Align.

El enrutado manual se inicia con Ctrl+W o click en un pad. Durante el enrutado se puede cambiar el modo de esquina con Shift+Space para alternar entre 45°, 90° y arco. Se termina la ruta con click derecho → End Route o presionando Esc.

Una característica distintiva de Altium es su motor de enrutado automático. Se accede desde Auto Route → All y se configuran las estrategias de enrutado. Los parámetros principales incluyen número de pasadas (intentos del algoritmo), prioridad de capas, cantidad de vías permitidas, y estrategias como Fanout (crea vías iniciales), Memory (usa resultados anteriores), Main (algoritmo principal), Recorner (optimiza esquinas) y Cleanup (limpia rutas finales).

PEl auto-routing es útil para evaluar rápidamente la viabilidad del layout y para diseños con muchas conexiones, aunque es importante revisar y optimizar las rutas generadas.

El Design Rule Check (DRC) es crucial antes de enviar a fabricación. Se ejecuta desde Tools → Design Rule Check y se configuran las reglas a verificar. El panel muestra violaciones con ubicación exacta, y con doble click en cada error se navega directo a la ubicación. Los errores comunes incluyen violaciones de clearance (pistas muy cerca), unrouted nets (conexiones sin completar) y violaciones de ancho mínimo (pistas muy delgadas).

Se configura el formato (2:5 ofrece suficiente precisión), las unidades (milímetros o pulgadas según fabricante) y se seleccionan todas las capas necesarias. Altium crea una carpeta con archivos .gbr Otros formatos disponibles incluyen DXF/DWG para importar en software CAD mecánico, PDF para documentación, IDF para validación mecánica, STEP para modelo 3D completo, y ODB++ como alternativa estándar a Gerber. Los archivos de perforación se generan desde File → Fabrication Outputs → NC Drill Files en formato Excellon.

Altium Designer incluye capacidades de simulación integradas que permiten verificar el comportamiento del circuito antes de fabricarlo. Se puede realizar análisis DC, AC transitorio y de frecuencia directamente desde el esquemático. Para simular, se configuran los parámetros de análisis desde Simulate → Setup y se ejecuta con Simulate → Run. Los resultados se visualizan en gráficas que muestran voltajes, corrientes y respuestas en frecuencia según el tipo de análisis seleccionado.
